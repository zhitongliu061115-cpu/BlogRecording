# 贡献者行为准则 (Code of Conduct)

## 我们的承诺

为了营造一个开放和热情的环境，作为本项目的贡献者和维护者，我们承诺致力于使参与我们项目和社区的每个人都能获得免受骚扰的体验，无论其年龄、体型、残疾、民族、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教信仰，或性取向如何。

我们承诺以有助于建设一个充满活力、健康和可持续的社区的方式行事和互动。

## 我们的标准

有助于创造积极社区环境的行为包括：

*   使用友好、包容的语言
*   尊重不同的观点和经验
*   优雅地接受建设性的批评
*   关注对社区整体最有利的事情
*   对社区其他成员展现同理心

不可接受的行为包括：

*   使用性化的语言或图像，以及不受欢迎的性关注或挑逗
*   挑衅、侮辱或贬损的评论，以及人身或政治攻击
*   公开或私下的骚扰
*   未经明确许可，发布他人的私人信息（例如物理地址或电子邮箱地址）
*   其他在专业环境中可被合理视为不适当的行为

## 执行责任

项目维护者有责任澄清可接受行为的标准，并应针对任何不适当、威胁、冒犯或有害的行为，采取公平且适当的纠正措施。

项目维护者有权、也有责任移除、编辑或拒绝违反本行为准则的评论 (comments)、提交 (commits)、代码 (code)、维基编辑 (wiki edits)、问题 (issues) 及其他贡献。维护者也可以暂时或永久地封禁任何他们认为从事不当、威胁、冒犯或有害行为的贡献者。

## 适用范围

本行为准则不仅适用于项目的所有代码仓库、Issue 区和讨论区，也适用于任何代表本项目在公共场合（如线上分享、社交媒体等）发声的个人。当社区成员在官方平台内外代表项目时，皆受此准则约束。

## 执行与举报

任何滥用、骚扰或其他不可接受的行为都可以通过发送邮件至 **3790764375@qq.com** 向项目维护团队举报。所有投诉都将得到审查和调查，并会产生被认为必要和适当的响应。项目维护团队有义务为举报者保密。

未能善意遵守或执行本行为准则的维护者，可能会面临由项目其他领导层成员决定的临时或永久的处罚。

## 执行指南

项目维护者将根据以下准则，对违反本准则的行为决定处罚结果：

1. **纠正 (Correction)**
   * **影响：** 在社区中造成了不专业或不受欢迎的影响。
   * **后果：** 维护者将发出书面警告，说明违规性质并要求停止该行为。可能需要公开道歉。

2. **警告 (Warning)**
   * **影响：** 发生单一的或轻微的违规行为。
   * **后果：** 警告并附带要求改正的后果说明。在规定的一段时间内，禁止与相关人员互动。

3. **临时封禁 (Temporary Ban)**
   * **影响：** 发生严重的违规行为，或持续发生不适当的行为。
   * **后果：** 在规定的一段时间内，禁止参与社区的任何互动（包括提交代码、评论 Issue 等）。

4. **永久封禁 (Permanent Ban)**
   * **影响：** 表现出系统性违反行为准则的模式，或单次行为极其恶劣。
   * **后果：** 永久禁止参与项目社区内的任何形式互动。

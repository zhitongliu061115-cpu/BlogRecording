工作流

四步完成一次播客复盘

01

录音 / 导入

支持系统内录、麦克风录音、本地媒体导入、URL 媒体导入。

02

本地转写

在本地做音频处理、VAD、SenseVoice 转写与说话人分离。

03

AI 总结

只把转写文本、时间戳、说话人标签发给 DeepSeek 做摘要、标签、高光和问答。

04

导出与追问

导出 Markdown / TXT / JSON，或围绕当前会话继续追问。

隐私边界

把敏感部分留在本地，把云端输入缩到最小

只上传文本，不上传原始音频

发送给 DeepSeek 的只有转写文本、时间戳和说话人标签，原始音频尽量留在本地。

API Key 本地加密保存

项目说明中明确写明 API Key 本地加密保存，避免把密钥硬编码进站点或仓库。

AI 总结不是强制步骤

没有联网或暂时不想调用模型时，你仍然可以录音、转写、整理和导出。

FAQ

先把最关键的问题回答掉

这是安卓吗？

是。它是 Android App，技术栈是 Kotlin + Jetpack Compose，最低 Android 版本为 29+。

需要联网吗？

录音、导入、本地转写相关流程以本地优先为主；如果要调用 DeepSeek 做总结或问答，则需要联网。

AI 总结是否可选？

可选。不要把它理解成“完全离线 AI 总结”，而是本地转写基础上的可选增强。

下载后怎么安装？

点击“立即下载 APK”后获取安装包，在 Android 设备上允许对应来源安装，然后完成安装即可。
